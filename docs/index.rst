beangulp: Importers Framework for Beancount
===========================================

.. title:: Overview

This library defines a workflow that helps you build importers for
your various accounts and provides utilities that work with them. It
works with the unreleased Beancount v3. For Beancount v2 use the
`beancount.ingest` framework.
